C++程序中内嵌git-commit-id

项目配置

  • 构建方式：visual studio 2017
  • 运行平台：win10
  • 运行方式：dll

项目需求

希望在dll中可以获得最近构建代码的git commit id，组成dll的版本号，方便管理与出现问题时定位原因。

解决方案

实现原理

graph LR
    start[开始build] --> prebuild((执行脚本))
    getGitIdScript(#include &apos;&apos;stdafx.h&apos;&apos;<br>extern const std::string gitCommitId=&apos;&apos;d8as8d&apos;&apos;) -- 生成 --- prebuild
    style getGitIdScript fill:#f4bc42,stroke:#777,stroke-width:3px,fill-opacity:0.5
    prebuild --> compile[编译]

之后就可以直接在dll代码中引入该头文件访问git id
